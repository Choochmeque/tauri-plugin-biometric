## Default Permission

This permission set configures which
biometric features are by default exposed.

#### Granted Permissions

It allows acccess to all biometric commands.

#### This default permission set includes the following:

- `allow-authenticate`
- `allow-status`
- `allow-has-data`
- `allow-get-data`
- `allow-set-data`
- `allow-remove-data`
